Fresh Fruit Store 
A full-stack e-commerce web application for fresh fruits and vegetables.

🛠️ Tech Stack
Frontend:
React 19.0.0
React Router DOM 7.4.0
Tailwind CSS 4.0.17
Vite 6.3.6

Backend:
Node.js
Express.js
MongoDB with Mongoose
JWT for authentication
Multer for file uploads

📦 Installation & Setup
Prerequisites
Node.js (v18 or higher)
MongoDB
Git
Client Setup
cd client
npm install
npm run dev
Server Setup
cd server
npm install
cp .env.example .env
Configure your environment variables
npm run dev
